Project Overview

This project simulates a modern data processing workflow commonly found in e-commerce environments, combining:

Synthetic data generation
Batch processing
Distributed computing
ETL pipelines
Data analytics
Data visualization
Real-time streaming

The goal is to demonstrate practical Data Engineering concepts using industry-standard tools and technologies.



Features
Data Ingestion

Comparison between Pandas and Dask for processing large JSON datasets and performing analytical operations.

Key concepts:

Batch file processing
Large dataset handling
Lazy evaluation with Dask
Aggregation and exploratory analysis
ETL Pipeline

Implementation of a complete ETL workflow using PySpark.

Main operations:

Data extraction from Parquet files
Dataset joins
Data transformation and cleaning
Aggregations
Partitioned data storage
Export to optimized Parquet format
Analytics & Reporting

Revenue analysis by product category with automated reporting and visualization.
